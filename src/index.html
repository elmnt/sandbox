Sandbox.

A lightweight front-end environment to quickly spin up a server, and start writing code.

Includes tachyons-sass

Some basic commands:

  1. npm run dev fires up your local server and gets you started.
  2. npm run clean removes the dist/ folder and your local build files.
  3. npm run ship builds the dist/ folder and copies all your compiled, production-ready code.
